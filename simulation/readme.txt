Instructions for launching the program

Prerequisites:
	- Import previously the .jar files of the other teams
	- Add the .jar files of the other teams to the project class path (right click on jar, classpath, add to classpath)

Launch instructions:
	- Double click on the file "Run Platform.launch"
	- Select "Run as" y luego "Run Platform"
	
Results:
	The results of the reporter will be available on the standard output console after the simulation ends, or in the file specifiec by the value "report.name" of the settings.properties file
